## üõ†Ô∏è How to use this template

### 1. Repository Setup
1. **Use this template**: Click the "Use this template" button on GitHub to create a new repository.
2. **Make it Public**: Your repository **must** be public for Giscus to work.
3. **Enable Discussions**: 
   - Go to **Settings** > **General**.
   - Scroll down to the **Features** section and check the **Discussions** box.

### 2. Install Giscus App
1. Go to [github.com/apps/giscus](https://github.com/apps/giscus).
2. Install the app and grant it access to your specific repository.

### 3. Configure the Script
1. Visit [giscus.app](https://giscus.app).
2. Enter your repository name (e.g., `YourUsername/YourRepo`).
3. Scroll down to the **Configuration** section to get your specific IDs.
4. Open `index.html` in your editor and update the `data-repo-id` and `data-category-id` values in the script at the bottom of the file.

### 4. Deploy
1. Go to **Settings** > **Pages**.
2. Set the source to the `main` branch and `/root` folder.
3. Save, and your blog-forum will be live!
